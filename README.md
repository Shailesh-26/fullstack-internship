# DecodeLabs Full Stack Internship – Task 1

## 📌 Project Overview
This project is part of the DecodeLabs Full Stack Development Internship (Industrial Training Kit – Project 1).

The objective was to build a responsive frontend interface using core web technologies without any frameworks.

---

## 🎯 Objective
To create a mobile-first responsive portfolio landing page using:

- Semantic HTML5
- CSS3 (Grid & Flexbox)
- Vanilla JavaScript

---

## 🧱 Features Implemented

- ✅ Mobile-first layout approach
- ✅ Responsive breakpoints (768px & 1024px)
- ✅ Semantic HTML structure
- ✅ CSS Grid for macro layout
- ✅ Flexbox for navigation and components
- ✅ Sticky header
- ✅ JavaScript mobile navigation toggle
- ✅ Clean and accessible UI design

---

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

No frameworks or external libraries were used.

---

## 📱 Responsive Design Strategy

The layout was built using a mobile-first approach and expanded using `min-width` media queries:

- Mobile: Default layout
- Tablet: 768px
- Desktop: 1024px
